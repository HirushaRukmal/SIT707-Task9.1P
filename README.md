# OnTrack TDD and CI Demo

This is a simple Maven Java project for the SIT707 TDD and CI task.

## Function implemented
View submitted task status for a simplified OnTrack-style workflow.

## Valid statuses
- PENDING
- DISCUSSION
- COMPLETE
- RESUBMIT

## Run tests locally
```bash
mvn clean test
```

## CI
GitHub Actions workflow is provided in `.github/workflows/ci.yml`.
